### 使用方式

到`main.py`第 8 行修改`list_serach_job`的陣列內容（可以做批量查詢），
改完存檔後 cmd 輸入：

```shell
python main.py
```

等一陣子你的爬蟲就出爐囉～（跟`main.py`同個目錄）
